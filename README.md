# Peng Zhao — Academic Website

Source for [pengzhaostat.github.io](https://pengzhaostat.github.io), the academic website of Peng Zhao, Assistant Professor of Statistics at the University of Delaware.

The site is built with [al-folio](https://github.com/alshedivat/al-folio) and deployed through GitHub Pages.

## Local development

```bash
npm ci
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Content

- Main pages are in `_pages/`.
- Publications are maintained in `_bibliography/papers.bib`.
- Announcements are in `_news/`.
- The downloadable CV is `assets/pdf/Peng_Zhao_CV.pdf`.
